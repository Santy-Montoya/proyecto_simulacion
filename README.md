# Proyecto Simulación y Computación Numérica (Python + Streamlit)

## Ejecutar
1. Crear entorno:
   - Windows: py -m venv .venv && .venv\Scripts\activate
   - Mac/Linux: python3 -m venv .venv && source .venv/bin/activate
2. Instalar:
   pip install -r requirements.txt
3. Correr:
   streamlit run app.py

## Métodos incluidos
- Lagrange
- Interpolación de Newton (diferencias divididas)
- Bisección
- Newton-Raphson
- Newton para sistemas no lineales (2 variables)
- Jacobi
- Gauss-Seidel
- Diferencias finitas (BVP 1D Poisson)
- Mínimos cuadrados (ajuste polinomial)
